# Limitations

## Model limitations

- The model is an ABP-type reduced numerical prototype.
- The internal attribute variable is abstract and does not represent a calibrated physical material, organism, robot, or social actor.
- The information feedback control term is a mathematical variance-minimization rule, not an experimentally implemented controller.
- The interaction potentials and parameters are model choices, not fitted experimental active-matter parameters.
- The finite-size range is limited to the tested N values.

## Validation limitations

- No experimental active-matter data are fitted.
- No hydrodynamic continuum limit is derived.
- No formal universality class is established.
- No rigorous theorem of kinetic arrest is proven.
- No biological, social, civilizational, or engineering system is validated.

## Repository limitations

- The original uploaded folder contained a large Python virtual environment, which is excluded from the GitHub body.
- This repository is a paper companion archive, not a complete virtual environment.
