## Purpose

This page gives technically interested first-time readers a compact, active-particle-specific visual map of the repository before they read the manuscript, figures, source materials, and claim-boundary documents.

It summarizes:

- the active-particle-specific mission variable,
- kinetic-arrest / noise-induced-annealing diagnosis,
- particle mobility, order, clustering/arrest, feedback, and noise channels,
- feedback-assisted escape versus artifact-like apparent recovery,
- reconstructed-public-reproducer provenance caution,
- evidence hierarchy,
- repository reading order,
- explicit material-design / experimental / industrial / deployment non-claims.

All graph sections include replay controls so the reader can step through the visual logic interactively.
